# Framewise UI Design Specification

## Screen 1: Camera (主取景页面)

```
┌──────────────────────────────────────────────┐
│ [⚙️]                              [92] [🔄] │  ← 顶部栏：设置/评分/翻转镜头
│                                              │
│                                              │
│                    ┌──┐                      │
│                    │  │ ← 细半透明三          │
│                    │  │   分法网格线          │
│                    │  │                      │
│              ──────┘  └────── ← 地平线      │
│                         ● ← 主体框           │
│                    ↗                         │  ← 方向箭头
│                                              │
│  ┌──────────────────────────────────────┐    │
│  │  ← 向右移动主体到交点    ← 稍 ↑ 降机位  │  │  ← 底部建议区
│  └──────────────────────────────────────┘    │
│                                              │
│  [📷]    [⏱️]    [ ⏺ ]    [🔦]    [🔍│]   │  ← 底部控制栏
│  图库    定时    快门    闪光灯   变焦滑块    │
└──────────────────────────────────────────────┘
```

### 布局参数
- 三分线: 1dp, 白色 30% 透明度
- 主体框: 2dp 虚线, 绿色/橙色/红色 按评分
- 方向箭头: 2dp 圆头线条 + V 头, 半透明
- 底部建议区: 暗底 85% 透明度, 圆角 20dp, 文字白色
- 评分 badge: 32dp 圆形, 右上角
- 控制栏: 底部安全区, 间距均匀

### 交互
- 点击取景区域: 对焦/测光
- 双指捏合: 变焦
- 右侧滑动: 变焦滑块
- 底部建议: 每 3s 自动轮换

## Color System

| Token | Color | Hex | Usage |
|-------|-------|-----|-------|
| ScorePerfect | Green | #3fb950 | Score ≥ 85, good composition |
| ScoreGood | Orange | #d29922 | Score 70-84, minor adjustments |
| ScoreBad | Red | #f85149 | Score < 70, needs major fix |
| BackgroundDark | Dark | #0d1117 | App background |
| SurfaceDark | Dark | #161b22 | Card/panel background |
| SurfaceDark2 | Dark | #1c2333 | Elevated surfaces |
| BorderDim | Gray | #2a313c | Subtle borders |
| TextWhite | White | #e6edf3 | Primary text |
| TextDim | Gray | #9aa6b2 | Secondary text |
| AccentBlue | Blue | #58a6ff | Interactive elements |
| TransparentGrid | White 20% | #33ffffff | Grid lines |

## Typography
- App: System default (Roboto on Android)
- Suggestion text: 14sp Medium
- Score badge: 16sp Bold
- Settings title: 16sp Medium
- Settings description: 13sp Regular
- Button labels: 14sp Medium
